# draw-tool
Draw vector elements online (mapbox-gl + mapbox-gl-draw + nebula.gl)
在地图中绘制编辑点线面等矢量要素

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
